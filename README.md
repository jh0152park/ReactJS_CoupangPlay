# ReactJS_CoupangPlay
CoupangPlay Clone coding to ReactJS study

# Main Feature
- Home: Default page, shown popular, top rated and other movies
- TV: shown Airing, On the Air, popular, top rated TV Shows
- Search: shown search result of `keyword`
